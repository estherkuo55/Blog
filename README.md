# Blog
Blog Management System

- Use Handlebars (HBS) as the template engine to create dynamic pages (such as blogs, posts, categories, etc.).
- Use Bootstrap for front-end UI design to achieve a responsive layout.
- Implement a user authentication mechanism (Login/Register/Session) to control the operation permissions of different users.
- Establish CRUD functions:
  - Post (Add, Display, Delete)
  - Category (Add, Manage, Delete)
- Establish a user login history function to record login time and device information.
